# AI Prompt Engineering Projects (IBM SkillsBuild)

## About This Repository
This repository contains my hands-on projects from the IBM SkillsBuild course "Craft Precise Prompts for AI Models".

The focus of these projects is to understand how to design effective prompts and create structured templates for generating accurate outputs using Large Language Models (LLMs).

## Certifications
- AI Fundamentals Guided Learning with Capstone – IBM SkillsBuild
- Craft Precise Prompts for AI Models – IBM SkillsBuild

## Project 1: Prompt Design for Specific Outputs
In this project, I created prompts to generate clear and structured responses from AI models.

### Example
**Prompt:**
Write a professional email for a job application.

**Output:**
Generated a structured professional email response using AI based on clear prompt instructions.

## Project 2: Prompt Templates Using Variables (In Progress)
